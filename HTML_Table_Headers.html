Vertical Table Headers

The first column becomes table headers if you set the first cell in each table row to a TH element:

Firstname AAAAA BBBBB
Lastname CCCCC DDDDD
Age 87 86


A header that spans two columns

Use the colspan attribute to have a header span over multiple columns.

Name Age
aaaaaa bbbbbb cccccc 666666 888888
dddddd eeeeee ffffff 555555 000000
gggggg hhhhhh iiiiii 888888 666666
