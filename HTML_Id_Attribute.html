The Id Attribute

Use CSS to style an element with the id "myheading":

The id Attribute specifies a unique id for an HTML element. the value of the id attribute must be unique within the html document.

The syntax for id is: write a has(#), followed by an id name. Then, define the CSS properties within curly braces{}.

The id Name is Case Sensitive!

The id name must contain at least one character, cannot start with a number, and must not contain whitespaces.
