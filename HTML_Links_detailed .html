Using Target Attribute.

the target attribute specifies where to open the linked document.

the target attribute can have one of the following values:

_self
_blank
_parent
_top

Using _self

Default. Open the document in the same window/tab as it was clicked.

Visit Google!

If target="_self",the link will open in a same window or tab.

Using _blank

Opens the document in a new window or tab.

Visit Instagram!

If target="_blank",the link will open in a new browser window or tab.

Using _parent

Opens the document in the parent frame.

Visit Canva!

Using _top

Opens the document in the full body of the window.

Visit Snapchat!
